# Combined Edit Plan — The Woman in My Phone

**Source:** V9 manuscript edit plan + query package review + structural gap analysis
**Goal:** Cut ~15% (87K → ~70-74K words). Humanize the voice. Fix query package inconsistencies. Ship a submission-ready manuscript + query package.
**Total estimated time:** 15-18 hours across all phases

## Phase 1: Query Package Fixes (90 minutes, do first)

These are independent of the manuscript edits and can be submitted immediately if you're querying the current draft.

### 1. Lock the word count

The query letter says 85,000. The query package says 58,700. The synopsis says 57,000. The structure targets 75,000. An agent who reads all three will wonder which draft you're actually sending.

**Action:** Pick the number that matches the manuscript you'd send if someone asked tomorrow. Update every file to that one number:

| File | Current | Fix |
|---|---|---|
| `QUERY-LETTER.md` | 85,000 | Update to actual count |
| `QUERY-PACKAGE.md` (header) | 58,700 | Same |
| `QUERY-PACKAGE.md` (Version A, B, C) | 57,000 | Same |
| `SYNOPSIS.md` (header) | 57,000 | Same |

If the manuscript is currently ~87K and you plan to cut to ~74K — don't query until the cut is done, or query the 87K version as-is with the correct number. Never pitch a word count you haven't written yet.

### 2. Trim the comp list from 6 to 3

Six comps dilutes the signal. Agents want 2-3 that say "this is my shelf space."

**Keep:**
- *Normal People* (Rooney) — intimacy through digital communication. Direct structural parallel.
- *Fleishman Is in Trouble* (Brodesser-Akner) — unreliable male narrator, marriage dissolution from inside. Exact tonal match.
- *Open Water* (Azumah Nelson) — long-distance love, cultural specificity, the weight of almost. Emotional register match.

**Cut:**
- *A Little Life* — wrong scale. 720 pages of relentless trauma vs. a 74K-word reckoning. Agents will think you're overscoping.
- *Exciting Times* — too niche, the connection is thin (digital-age love triangle is a surface-level match).
- *Such a Fun Age* — the "you're the villain" parallel is real but the books are tonally different enough that it confuses more than it clarifies.

**Action:** In `QUERY-LETTER.md`, cut the sentence to: "sits alongside *Normal People*, *Fleishman Is in Trouble*, and *Open Water*." In `QUERY-PACKAGE.md`, remove the three cut titles from the comparable titles table.

### 3. Soften "based on real events"

"Based on real events" makes some agents worry the author hasn't done the novelistic work. The Salesforce line and "the shame is entirely real" already do the heavy lifting — they signal authenticity without claiming memoir.

**Action:** In the "About me" paragraph of `QUERY-LETTER.md`, change:

> This novel is based on real events. The names and some details have been changed. The artifacts are reconstructed from memory and records. The shame is entirely real.

To:

> This novel is inspired by lived experience. The artifacts are reconstructed from memory and imagination. The shame is entirely real.

Three sentences instead of four. "Inspired by lived experience" signals the same thing without inviting the "is this a memoir?" question. "Memory and imagination" replaces "memory and records" — the word "records" makes it sound like court documents, not fiction.

The personalized query versions (A, B, C) already use "writing from lived experience" — that's the right framing. No changes needed there.

### 4. Reconsider the dedication placement

The current dedication: *"For everyone who's been the villain of someone else's story and didn't know it yet."*

The problem: it announces the book's pivot before page one. The reader arrives at Ari's self-awareness before he does. The whole engine of the novel is the gap between what Ari thinks he's doing (loving Meera) and what he's actually doing (being the villain of two women's stories). If the dedication closes that gap upfront, the first 150 pages lose tension.

**Two options:**

**(a) Move it to the end.** Make it an author's note after the last chapter. The reader has the full journey. Now the line lands as a confession, not a spoiler.

**(b) Replace it with something from the ending's emotional register** — the arrival, not the reckoning. Something like: *"For everyone who learned to sit on the throne alone."* That's the King chapter's thesis. It points forward without revealing what the character has to lose first.

**Action:** This one's a judgment call, not a clear fix. Try both options and see which one feels right when you read the opening pages cold. Not urgent for the query package — the dedication doesn't go in the query letter anyway.

## Phase 3: Global Pattern Sweep (3-4 hours)

Run these across the full manuscript before chapter-level edits. They're mechanical and compound — doing them first means the chapter edits are working on cleaner material.

### 8. "Not X. Not Y. Z." construction

Appears 30+ times. Reduce by 70% — keep ~10 of the strongest, cut the rest.

**Example to cut:** "Not a hug. Not the polite kind... I HELD her." → Keep maybe 30% of these. Kill the rest.

**Example to keep:** Whichever 10 genuinely earn the construction by creating a contrast that couldn't be expressed any other way.

### 9. "That's..." sentences after moments

Cut 50%. These are explanations posing as insights.

**Example to cut:** "That's my marriage in a screenshot." — The screenshot already IS the marriage. Don't label it.

**Example to keep:** The ones that genuinely add information the reader doesn't already have.

### 10. Aphorisms / tweetable lines

Cap at 2-3 per chapter. Cut the rest, especially at emotional peaks where they break the spell.

**Example to cut:** "Theories are just loneliness wearing a lab coat." → "I had theories. None of them helped."

**Rule:** If the line sounds good as a tweet, it's probably not earning its place in the scene. Prose that reads like a caption is prose performing for an audience.

### 11. "I know how it sounds" / "I need you to know" / direct reader address

Cut 80%. Keep 2-3 instances across the entire book — not 2-3 per chapter.

Direct address works when it's rare. When it's frequent, it becomes the narrator performing self-awareness instead of demonstrating it.

### 12. Explaining after showing

Every time a scene or dialogue exchange makes the point and is followed by a paragraph restating it, cut the restatement. This is the single most common pattern to fix.

**Test:** Read the dialogue/scene, then cover the explanation paragraph. Did you lose anything? If not, cut it.
